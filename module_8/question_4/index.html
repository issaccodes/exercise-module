Q4. Create an ordered list of HTML tags. Each list item must include the tag name and some information about the tag.
  1. <html> – Defines the root of an HTML document. It contains all the other elements of the page.
  2. <head> – Contains meta-information about the document, like its title, character set, and linked resources (like stylesheets).
  3. <title> – Specifies the title of the webpage, which appears in the browser's title bar or tab.
  4. <body> – Represents the content of the document that is visible to users, like text, images, and links.
  5. <h1> to <h6> – Define headings, with <h1> being the highest (largest) level and <h6> being the lowest (smallest).
  6. <p> – Defines a paragraph of text.
  7. <a> – Defines a hyperlink, allowing users to click and navigate to another page or resource.
  8. <img> – Embeds an image into the webpage. The "src" attribute specifies the path to the image file.
  9. <ul> – Defines an unordered list of items, typically rendered as bullet points.
  10. <ol> – Defines an ordered list, where items are numbered.
  11. <li> – Defines a list item in both ordered (<ol>) and unordered (<ul>) lists.
  12. <div> – Defines a division or section in the document. It's used as a container for HTML elements and is commonly used with CSS to style sections.
  13. <span> – Used to group inline elements in a document. It's a generic container for text that can be styled with CSS.
  14. <form> – Defines an HTML form used for collecting user input.
  15. <input> – Defines an input field where users can enter data. It can be of different types, such as text, password, email, etc.
