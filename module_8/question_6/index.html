1. MongoDB

    Primary Use Cases Key Features/Benefits
    Storing data in flexible, JSON-Like documents for
    applications with unstructured or semi-structured data.
    • NoSQL databse
    • Scalable and flexible data model
    • High availability abd easy replication

  2. Express.js

    Primary Use Cases Key Features/Benefits
    Building web servers and APIs for handling HTTP
    requests and responses in Node.js applications.
    • Minimal and flexible web framework
    • Middleware support for handling requests
    • Simplifies routing and server management

  3. React

    Primary Uses Cases Key Feature/Benefits
    Building dynamic and interactive user interface,
    especially for single-page applications.
    • Component-based architecture
    • Virtual DOM for fast rendering
    • Supports server-side rendering and SEO

  4. Node.js

    Primary Use Cases Key Features/Benifits
    Running javascript on the server to handle backend
    logic, file I/O, and real-time data processing.
    • Asynchronous and non-blocking I/O
    • Scalable and lightweight server enviroment
    • Large ecosystem of packages via npm
